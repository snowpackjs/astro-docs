Astroプロジェクトは、フロントエンド開発者向けに設計されたシングルコマンドのWeb公開プラットフォームである[Surge](https://surge.sh/)にデプロイすることができます。

## デプロイ方法

1. [Surge CLI](https://www.npmjs.com/package/surge)をまだグローバルにインストールしていない場合、インストールします。

    ```shell
    npm install -g surge
    ```

2. プロジェクトのルートディレクトリからAstroサイトをビルドします。

    ```shell
    npm run build
    ```

3. CLIを使用してSurgeにデプロイします。

    ```shell
    surge dist
    ```

    デプロイ時に `surge dist yourdomain.com` を実行すると、[カスタムドメイン](http://surge.sh/help/adding-a-custom-domain) を使用することができます。
