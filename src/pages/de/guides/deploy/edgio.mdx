Sie können Ihr Astro-Projekt auf [Edgio](https://www.edg.io/) veröffentlichen, einer Edge- und CDN-Plattform zum Veröffentlichen, Schützen und Beschleunigen von Websites und APIs.

:::tip
Schauen Sie sich den [Astro-Leitfaden in den Dokumentationen von Edgio an](https://docs.edg.io/guides/astro)!
:::

## Wie veröffentliche ich

1. Installieren Sie die [the Edgio CLI](https://docs.edg.io/guides/cli) global über die Befehlszeile, falls noch nicht geschehen.

    ```bash
    npm install -g @edgio/cli
    ```

2. Fügen Sie Edgio Ihrer Astro-Seite hinzu.

    ```bash
    edgio init
    ```

3. (Optional) Aktivieren Sie das Serverseitiges Rendering

    Nachdem Sie [@astrojs/node with Astro](/de/guides/integrations-guide/node/), konfiguriert haben, geben Sie den Pfad der Serverdatei in der Datei `edgio.config.js` an, wie unten gezeigt:

    ```js ins={2,5-9}
    // edgio.config.js
    import { join } from 'path'

    module.exports = {
      astro: {
        // Der Pfad des Standalone-Servers, der Astro SSR ausführt.
        // Die Abhängigkeiten für diese Datei werden automatisch gebündelt.
        appPath: join(process.cwd(), 'dist', 'server', 'entry.mjs'),
      },
    };
    ```

4. Veröffentlichen Sie auf Edgio

    ```bash
    edgio deploy
    ```
