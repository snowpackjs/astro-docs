A referência a seguir é uma lista completa dos erros que você pode encontrar enquanto utiliza Astro. Para mais ajuda, incluindo pegadinhas comuns, por favor também veja nosso [Guia de Solução de Problemas](/pt-br/guides/troubleshooting/).

## Erros do Astro

- [**StaticRedirectNotAvailable**](/pt-br/reference/errors/static-redirect-not-available/) (E03001)<br/>`Astro.redirect` is not available in static mode.
- [**ClientAddressNotAvailable**](/pt-br/reference/errors/client-address-not-available/) (E03002)<br/>`Astro.clientAddress` is not available in current adapter.
- [**StaticClientAddressNotAvailable**](/pt-br/reference/errors/static-client-address-not-available/) (E03003)<br/>`Astro.clientAddress` is not available in static mode.
- [**NoMatchingStaticPathFound**](/pt-br/reference/errors/no-matching-static-path-found/) (E03004)<br/>No static path found for requested path.
- [**OnlyResponseCanBeReturned**](/pt-br/reference/errors/only-response-can-be-returned/) (E03005)<br/>Invalid type returned by Astro page.
- [**MissingMediaQueryDirective**](/pt-br/reference/errors/missing-media-query-directive/) (E03006)<br/>Missing value for `client:media` directive.
- [**NoMatchingRenderer**](/pt-br/reference/errors/no-matching-renderer/) (E03007)<br/>No matching renderer found.
- [**NoClientEntrypoint**](/pt-br/reference/errors/no-client-entrypoint/) (E03008)<br/>No client entrypoint specified in renderer.
- [**NoClientOnlyHint**](/pt-br/reference/errors/no-client-only-hint/) (E03009)<br/>Missing hint on `client:only` directive.
- [**InvalidGetStaticPathParam**](/pt-br/reference/errors/invalid-get-static-path-param/) (E03010)<br/>Invalid value returned by a `getStaticPaths` path.
- [**InvalidGetStaticPathsReturn**](/pt-br/reference/errors/invalid-get-static-paths-return/) (E03011)<br/>Invalid value returned by getStaticPaths.
- [**GetStaticPathsRemovedRSSHelper**](/pt-br/reference/errors/get-static-paths-removed-rsshelper/) (E03012)<br/>getStaticPaths RSS helper is not available anymore.
- [**GetStaticPathsExpectedParams**](/pt-br/reference/errors/get-static-paths-expected-params/) (E03013)<br/>Missing params property on `getStaticPaths` route.
- [**GetStaticPathsInvalidRouteParam**](/pt-br/reference/errors/get-static-paths-invalid-route-param/) (E03014)<br/>Invalid value for `getStaticPaths` route parameter.
- [**GetStaticPathsRequired**](/pt-br/reference/errors/get-static-paths-required/) (E03015)<br/>`getStaticPaths()` function required for dynamic routes.
- [**ReservedSlotName**](/pt-br/reference/errors/reserved-slot-name/) (E03016)<br/>Invalid slot name.
- [**NoAdapterInstalled**](/pt-br/reference/errors/no-adapter-installed/) (E03017)<br/>Cannot use Server-side Rendering without an adapter.
- [**NoMatchingImport**](/pt-br/reference/errors/no-matching-import/) (E03018)<br/>No import found for component.
- [**InvalidPrerenderExport**](/pt-br/reference/errors/invalid-prerender-export/) (E03019)<br/>Invalid prerender export.
- [**InvalidComponentArgs**](/pt-br/reference/errors/invalid-component-args/) (E03020)<br/>Invalid component arguments.
- [**PageNumberParamNotFound**](/pt-br/reference/errors/page-number-param-not-found/) (E03021)<br/>Page number param not found.
- [**FailedToLoadModuleSSR**](/pt-br/reference/errors/failed-to-load-module-ssr/) (E04001)<br/>Could not import file.
- [**InvalidGlob**](/pt-br/reference/errors/invalid-glob/) (E04002)<br/>Invalid glob pattern.
