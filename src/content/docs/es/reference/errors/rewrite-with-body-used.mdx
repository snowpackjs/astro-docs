> **RewriteWithBodyUsed**: Astro.rewrite() no se puede usar si el cuerpo de la solicitud ya ha sido leído. Si necesitas leer el cuerpo, primero clona la solicitud.

## ¿Qué salió mal?
`Astro.rewrite()` no se puede usar si el cuerpo de la solicitud ya ha sido leído. Si necesitas leer el cuerpo, primero clona la solicitud. Por ejemplo:

```js
const data = await Astro.request.clone().formData();

Astro.rewrite("/target")
```
