## ¿Adónde vamos?

En esta unidad, crearás plantillas para compartir elementos y estilos comunes en tus páginas y posts de blog.

Para ello:

- Crear componentes de plantillas reutilizables
- Pasa contenido a tus plantillas con `<slot />`
- Pasa datos de Markdown frontmatter a tus plantillas
- Varias plantillas anidadas
