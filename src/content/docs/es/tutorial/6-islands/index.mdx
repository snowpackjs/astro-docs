## ¿Adónde vamos?

En esta unidad, utilizarás **Astro islands** para introducir componentes de frontend framework en tu sitio de Astro. 

Aprenderás a:
- Añadeir framework de UI, Preact, a tu proyecto de Astro
- Utilizar Preact para crear un componente de saludo interactivo
- Saber cuándo _no_ elegir islas para la interactividad
