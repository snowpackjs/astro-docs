Aceptamos contribuciones de cualquier tamaño y contribuyentes de cualquier nivel de habilidad. Como un proyecto de código abierto, creemos en devolver a nuestros contribuyentes. Estamos encantados de ayudar con orientación sobre PR, escritura técnica y convertir cualquier idea de función en realidad.

## Maneras de contribuir

Hay muchas formas de contribuir al sitio web de la documentación de Astro!

El sitio web de la documentación de Astro es ... ¡un sitio web de Astro! Mantenerlo requiere no solo contenido escrito, sino también mantener el código de Astro y abordar preocupaciones de a11y, CSS, UI y UX. También hacemos que nuestra documentación esté disponible en varios idiomas, por lo que necesitamos ayuda para traducir todo el sitio.

¡También puedes hacer una contribución enorme al involucrarte dejando comentarios de revisión en PR, agregando ideas en problemas y discusiones existentes de GitHub y participando en nuestras tareas de mantenimiento de problemas "Fijados"!

Cada PR, especialmente los PR de traducción, ¡necesita evaluadores! Revisar PR y dejar comentarios, sugerencias o un aprobador "LGTM!" ("¡Me parece bien!") Es una gran manera de comenzar en Team Docs y aprender más sobre Astro.
