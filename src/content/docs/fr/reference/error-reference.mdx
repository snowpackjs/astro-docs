La référence suivante est une liste complète des erreurs que vous pouvez rencontrer lors de l'utilisation d'Astro. Pour une assistance supplémentaire, y compris les pièges courants, veuillez également consulter notre [Guide de dépannage](/fr/guides/troubleshooting/).

## Erreurs d'Astro

- [**UnknownCompilerError**](/fr/reference/errors/unknown-compiler-error/)<br/>Unknown compiler error.
- [**StaticRedirectNotAvailable**](/fr/reference/errors/static-redirect-not-available/)<br/>`Astro.redirect` is not available in static mode.
- [**ClientAddressNotAvailable**](/fr/reference/errors/client-address-not-available/)<br/>`Astro.clientAddress` is not available in current adapter.
- [**StaticClientAddressNotAvailable**](/fr/reference/errors/static-client-address-not-available/)<br/>`Astro.clientAddress` is not available in static mode.
- [**NoMatchingStaticPathFound**](/fr/reference/errors/no-matching-static-path-found/)<br/>No static path found for requested path.
- [**OnlyResponseCanBeReturned**](/fr/reference/errors/only-response-can-be-returned/)<br/>Invalid type returned by Astro page.
- [**MissingMediaQueryDirective**](/fr/reference/errors/missing-media-query-directive/)<br/>Missing value for `client:media` directive.
- [**NoMatchingRenderer**](/fr/reference/errors/no-matching-renderer/)<br/>No matching renderer found.
- [**NoClientEntrypoint**](/fr/reference/errors/no-client-entrypoint/)<br/>No client entrypoint specified in renderer.
- [**NoClientOnlyHint**](/fr/reference/errors/no-client-only-hint/)<br/>Missing hint on `client:only` directive.
- [**InvalidGetStaticPathParam**](/fr/reference/errors/invalid-get-static-path-param/)<br/>Invalid value returned by a `getStaticPaths` path.
- [**InvalidGetStaticPathsReturn**](/fr/reference/errors/invalid-get-static-paths-return/)<br/>Invalid value returned by getStaticPaths.
- [**GetStaticPathsRemovedRSSHelper**](/fr/reference/errors/get-static-paths-removed-rsshelper/)<br/>getStaticPaths RSS helper is not available anymore.
- [**GetStaticPathsExpectedParams**](/fr/reference/errors/get-static-paths-expected-params/)<br/>Missing params property on `getStaticPaths` route.
- [**GetStaticPathsInvalidRouteParam**](/fr/reference/errors/get-static-paths-invalid-route-param/)<br/>Invalid value for `getStaticPaths` route parameter.
- [**GetStaticPathsRequired**](/fr/reference/errors/get-static-paths-required/)<br/>`getStaticPaths()` function required for dynamic routes.
- [**ReservedSlotName**](/fr/reference/errors/reserved-slot-name/)<br/>Invalid slot name.
- [**NoAdapterInstalled**](/fr/reference/errors/no-adapter-installed/)<br/>Cannot use Server-side Rendering without an adapter.
- [**NoMatchingImport**](/fr/reference/errors/no-matching-import/)<br/>No import found for component.
- [**InvalidPrerenderExport**](/fr/reference/errors/invalid-prerender-export/)<br/>Invalid prerender export.
- [**InvalidComponentArgs**](/fr/reference/errors/invalid-component-args/)<br/>Invalid component arguments.
- [**PageNumberParamNotFound**](/fr/reference/errors/page-number-param-not-found/)<br/>Page number param not found.
- [**ImageMissingAlt**](/fr/reference/errors/image-missing-alt/)<br/>Missing alt property.
- [**InvalidImageService**](/fr/reference/errors/invalid-image-service/)<br/>Error while loading image service.
- [**MissingImageDimension**](/fr/reference/errors/missing-image-dimension/)<br/>Missing image dimensions
- [**UnsupportedImageFormat**](/fr/reference/errors/unsupported-image-format/)<br/>Unsupported image format
- [**PrerenderDynamicEndpointPathCollide**](/fr/reference/errors/prerender-dynamic-endpoint-path-collide/)<br/>Prerendered dynamic endpoint has path collision.
- [**ExpectedImage**](/fr/reference/errors/expected-image/)<br/>Expected src to be an image.
- [**ExpectedImageOptions**](/fr/reference/errors/expected-image-options/)<br/>Expected image options.
- [**MarkdownImageNotFound**](/fr/reference/errors/markdown-image-not-found/)<br/>Image not found.
- [**ResponseSentError**](/fr/reference/errors/response-sent-error/)<br/>Unable to set response.
- [**MiddlewareNoDataOrNextCalled**](/fr/reference/errors/middleware-no-data-or-next-called/)<br/>The middleware didn't return a response or call `next`.
- [**MiddlewareNotAResponse**](/fr/reference/errors/middleware-not-aresponse/)<br/>The middleware returned something that is not a `Response` object.
- [**LocalImageUsedWrongly**](/fr/reference/errors/local-image-used-wrongly/)<br/>ESM imported images must be passed as-is.
- [**LocalsNotAnObject**](/fr/reference/errors/locals-not-an-object/)<br/>Value assigned to `locals` is not accepted.
- [**AstroGlobUsedOutside**](/fr/reference/errors/astro-glob-used-outside/)<br/>Astro.glob() used outside of an Astro file.
- [**AstroGlobNoMatch**](/fr/reference/errors/astro-glob-no-match/)<br/>Astro.glob() did not match any files.
- [**RedirectWithNoLocation**](/fr/reference/errors/redirect-with-no-location/)<br/>A redirect must be given a location with the `Location` header.
- [**InvalidDynamicRoute**](/fr/reference/errors/invalid-dynamic-route/)<br/>Invalid dynamic route.
- [**UnknownViteError**](/fr/reference/errors/unknown-vite-error/)<br/>Unknown Vite Error.
- [**FailedToLoadModuleSSR**](/fr/reference/errors/failed-to-load-module-ssr/)<br/>Could not import file.
- [**InvalidGlob**](/fr/reference/errors/invalid-glob/)<br/>Invalid glob pattern.
- [**FailedToFindPageMapSSR**](/fr/reference/errors/failed-to-find-page-map-ssr/)<br/>Astro couldn't find the correct page to render
