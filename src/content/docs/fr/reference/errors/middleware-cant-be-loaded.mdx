> **MiddlewareCantBeLoaded**: Le middleware a provoqué une erreur pendant qu'Astro essayait de le charger.

## Qu'est-ce qui n'a pas fonctionné ?
Lancé en mode développement lorsque le middleware génère une erreur en essayant de le charger.

Par exemple :
```ts
import {defineMiddleware} from "astro:middleware";
throw new Error("Error thrown while loading the middleware.")
export const onRequest = defineMiddleware(() => {
  return "string"
});
```
