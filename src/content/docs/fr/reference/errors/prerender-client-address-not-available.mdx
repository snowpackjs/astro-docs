> **PrerenderClientAddressNotAvailable**: `Astro.clientAddress` cannot be used inside prerendered routes

## Qu'est-ce qui a mal tourné ?
`Astro.clientAddress` ne peut pas être utilisé dans les itinéraires pré-rendus.
