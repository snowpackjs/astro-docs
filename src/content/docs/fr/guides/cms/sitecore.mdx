[Sitecore Experience Manager (XM)](https://www.sitecore.com/products/experience-manager) est un système de gestion de contenu d'entreprise construit sur ASP.NET.

### Prérequis

1. [Créez un site Web Sitecore Headless](https://doc.sitecore.com/xp/en/developers/sxa/103/sitecore-experience-accelerator/create-a-headless-tenant-and-site.html) en suivant la documentation officielle de Sitcore.
2. Exécutez la commande d'initialisation de projet suivante dans votre terminal :
  ```shell
  npx @astro-sitecore-jss/create-astro-sitecore-jss@latest
  ```
3. Suivez les instructions dans le terminal pour créer votre projet.
