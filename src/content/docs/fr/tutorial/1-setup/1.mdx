## Les outils de développement dont vous avez besoin

### Terminal

Vous utiliserez une **ligne de commande (terminal)** pour créer votre projet Astro et exécuter des commandes clés pour construire, développer et tester votre site.

Vous pouvez accéder à la ligne de commande par le biais d'un programme de terminal local pour votre système d'exploitation. Les applications courantes sont  **Terminal** (MacOS/Linux), **Command Prompt** (Windows), and **Termux** (Android). L'une de ces applications se trouve probablement déjà sur votre machine.

### Node.js

Pour qu'Astro fonctionne sur votre système, vous devez également avoir installé [**Node.js**](https://nodejs.org/en/), version `v18.14.1` ou ultérieure.

Pour vérifier si vous avez déjà installé une version compatible, exécutez la commande suivante dans votre terminal :

```sh
node -v

// Exemple sortie
v18.14.1
```

Si la commande renvoie un numéro de version supérieur à `v18.14.1`, vous êtes prêt !

Si la commande renvoie un message d'erreur tel que `Command 'node' not found`, ou un numéro de version inférieur à `v18.14.1`, vous devez [installer une version compatible de Node.js](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

### Editeur de Code

En plus, vous devrez télécharger et installer un **editeur de code** pour écrire votre code.

:::tip[Nous utiliserons...]
pour écrire votre code. **VS Code**, mais vous pouvez utiliser n'importe quel éditeur pour votre système d'exploitation.
:::

1. [Télécharger et installer VS Code](https://code.visualstudio.com/#alt-downloads) ou un autre éditeur de code de votre choix.
