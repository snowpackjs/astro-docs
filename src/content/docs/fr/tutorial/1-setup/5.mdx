Ici, vous connecterez votre dépôt GitHub à Netlify. Netlify utilisera ce projet pour construire et déployer votre site en direct sur le web à chaque fois que vous modifiez votre code.

:::tip[nous utiliserons...]
Ce tutoriel utilise Netlify, mais vous pouvez utiliser le service d'hébergement de votre choix pour déployer votre site sur l'internet.
:::

## Créer un nouveau site Netlify

1. Créez un compte gratuit sur [Netlify](https://netlify.com) si vous n'en avez pas.

    Notez votre nom d'utilisateur. Vous pourrez consulter votre tableau de bord et tous les sites que vous aurez créés à l'adresse suivante `https://app.netlify.com/teams/username`

2. Cliquez <kbd>Add new site</kbd> > <kbd>Import an existing project</kbd>.

    Il vous sera demandé de vous connecter à un fournisseur Git. Choisissez GitHub et suivez les étapes à l'écran pour authentifier votre compte GitHub. Ensuite, choisissez le dépôt GitHub de votre projet Astro dans la liste fournie.

3. A la dernière étape, Netlify vous montrera les paramètres du site de votre application. Les paramètres par défaut devraient être corrects pour votre projet Astro, vous pouvez donc faire défiler l'écran et cliquer sur <kbd>Deploy site</kbd>.

Félicitations, vous avez un site web Astro !

## Changer le nom de votre projet

Sur la page de présentation de votre site dans Netlify, vous verrez votre nom de projet généré aléatoirement et l'URL de votre site web sous la forme `https://project-name-123456.netlify.app`. Vous pouvez changer le nom de votre projet en quelque chose de plus mémorable, ce qui mettra automatiquement à jour votre URL.

## Visitez votre nouveau site web

Cliquez sur l'URL dans les paramètres de votre site ou tapez-la dans une fenêtre de navigateur pour consulter votre nouveau site web.
