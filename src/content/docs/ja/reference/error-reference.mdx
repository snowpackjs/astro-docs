以下のリファレンスは、Astroを使用している際に遭遇する可能性のあるエラーの完全なリストです。よくある落とし穴など、その他のサポートが必要であれば、[トラブルシューティングガイド](/ja/guides/troubleshooting/)も参照してください。

## Astroエラー

- [**UnknownCompilerError**](/ja/reference/errors/unknown-compiler-error/) (E01000)<br/>Unknown compiler error.
- [**StaticRedirectNotAvailable**](/ja/reference/errors/static-redirect-not-available/) (E03001)<br/>`Astro.redirect` is not available in static mode.
- [**ClientAddressNotAvailable**](/ja/reference/errors/client-address-not-available/) (E03002)<br/>`Astro.clientAddress` is not available in current adapter.
- [**StaticClientAddressNotAvailable**](/ja/reference/errors/static-client-address-not-available/) (E03003)<br/>`Astro.clientAddress` is not available in static mode.
- [**NoMatchingStaticPathFound**](/ja/reference/errors/no-matching-static-path-found/) (E03004)<br/>No static path found for requested path.
- [**OnlyResponseCanBeReturned**](/ja/reference/errors/only-response-can-be-returned/) (E03005)<br/>Invalid type returned by Astro page.
- [**MissingMediaQueryDirective**](/ja/reference/errors/missing-media-query-directive/) (E03006)<br/>Missing value for `client:media` directive.
- [**NoMatchingRenderer**](/ja/reference/errors/no-matching-renderer/) (E03007)<br/>No matching renderer found.
- [**NoClientEntrypoint**](/ja/reference/errors/no-client-entrypoint/) (E03008)<br/>No client entrypoint specified in renderer.
- [**NoClientOnlyHint**](/ja/reference/errors/no-client-only-hint/) (E03009)<br/>Missing hint on `client:only` directive.
- [**InvalidGetStaticPathParam**](/ja/reference/errors/invalid-get-static-path-param/) (E03010)<br/>Invalid value returned by a `getStaticPaths` path.
- [**InvalidGetStaticPathsReturn**](/ja/reference/errors/invalid-get-static-paths-return/) (E03011)<br/>Invalid value returned by getStaticPaths.
- [**GetStaticPathsRemovedRSSHelper**](/ja/reference/errors/get-static-paths-removed-rsshelper/) (E03012)<br/>getStaticPaths RSS helper is not available anymore.
- [**GetStaticPathsExpectedParams**](/ja/reference/errors/get-static-paths-expected-params/) (E03013)<br/>Missing params property on `getStaticPaths` route.
- [**GetStaticPathsInvalidRouteParam**](/ja/reference/errors/get-static-paths-invalid-route-param/) (E03014)<br/>Invalid value for `getStaticPaths` route parameter.
- [**GetStaticPathsRequired**](/ja/reference/errors/get-static-paths-required/) (E03015)<br/>`getStaticPaths()` function required for dynamic routes.
- [**ReservedSlotName**](/ja/reference/errors/reserved-slot-name/) (E03016)<br/>Invalid slot name.
- [**NoAdapterInstalled**](/ja/reference/errors/no-adapter-installed/) (E03017)<br/>Cannot use Server-side Rendering without an adapter.
- [**NoMatchingImport**](/ja/reference/errors/no-matching-import/) (E03018)<br/>No import found for component.
- [**InvalidPrerenderExport**](/ja/reference/errors/invalid-prerender-export/) (E03019)<br/>Invalid prerender export.
- [**InvalidComponentArgs**](/ja/reference/errors/invalid-component-args/) (E03020)<br/>Invalid component arguments.
- [**PageNumberParamNotFound**](/ja/reference/errors/page-number-param-not-found/) (E03021)<br/>Page number param not found.
- [**ImageMissingAlt**](/ja/reference/errors/image-missing-alt/) (E03022)<br/>Missing alt property.
- [**InvalidImageService**](/ja/reference/errors/invalid-image-service/) (E03023)<br/>Error while loading image service.
- [**MissingImageDimension**](/ja/reference/errors/missing-image-dimension/) (E03024)<br/>Missing image dimensions
- [**UnsupportedImageFormat**](/ja/reference/errors/unsupported-image-format/) (E03025)<br/>Unsupported image format
- [**PrerenderDynamicEndpointPathCollide**](/ja/reference/errors/prerender-dynamic-endpoint-path-collide/) (E03026)<br/>Prerendered dynamic endpoint has path collision.
- [**ExpectedImage**](/ja/reference/errors/expected-image/) (E03027)<br/>Expected src to be an image.
- [**ExpectedImageOptions**](/ja/reference/errors/expected-image-options/) (E03028)<br/>Expected image options.
- [**MarkdownImageNotFound**](/ja/reference/errors/markdown-image-not-found/) (E03029)<br/>Image not found.
- [**ResponseSentError**](/ja/reference/errors/response-sent-error/) (E03030)<br/>Unable to set response.
- [**MiddlewareNoDataOrNextCalled**](/ja/reference/errors/middleware-no-data-or-next-called/) (E03031)<br/>The middleware didn't return a response or call `next`.
- [**MiddlewareNotAResponse**](/ja/reference/errors/middleware-not-aresponse/) (E03032)<br/>The middleware returned something that is not a `Response` object.
- [**LocalsNotAnObject**](/ja/reference/errors/locals-not-an-object/) (E03033)<br/>Value assigned to `locals` is not accepted.
- [**AstroGlobUsedOutside**](/ja/reference/errors/astro-glob-used-outside/) (E03035)<br/>Astro.glob() used outside of an Astro file.
- [**AstroGlobNoMatch**](/ja/reference/errors/astro-glob-no-match/) (E03036)<br/>Astro.glob() did not match any files.
- [**RedirectWithNoLocation**](/ja/reference/errors/redirect-with-no-location/) (E03037)<br/>A redirect must be given a location with the `Location` header.
- [**InvalidDynamicRoute**](/ja/reference/errors/invalid-dynamic-route/) (E03038)<br/>Invalid dynamic route.
- [**UnknownViteError**](/ja/reference/errors/unknown-vite-error/) (E04000)<br/>Unknown Vite Error.
- [**FailedToLoadModuleSSR**](/ja/reference/errors/failed-to-load-module-ssr/) (E04001)<br/>Could not import file.
- [**InvalidGlob**](/ja/reference/errors/invalid-glob/) (E04002)<br/>Invalid glob pattern.
