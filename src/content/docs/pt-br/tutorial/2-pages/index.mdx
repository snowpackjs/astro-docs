Agora que você tem um site funcional na web, vamos adicionar páginas e postagens!

## Onde estamos agora?

Você pode encontrar o código para este ponto no tutorial no [GitHub](https://github.com/withastro/blog-tutorial-demo/tree/unit-2/start) ou
[StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/unit-2/start?file=src/pages/index.astro).

Compare seu código contra esse exemplo, ou se você acabou de se juntar a nós, faça um fork no StackBlitz para começar a codificar junto do seu navegador a partir de agora.

## Para onde vamos?

Nesta unidade, você irá
- Criar suas primeiras páginas Astro com a sintaxe `.astro`
- Adicionar postagens ao blog com arquivos Markdown (`.md`)
- Estilizar uma página individual com `<style>` 
- Aplicar estilos globais entre páginas

 Durante o caminho, você aprenderá como as **duas seções de um arquivo `.astro`**  funcionam juntas para criar uma página, e como utilizar variáveis e renderização condicional em suas páginas.
