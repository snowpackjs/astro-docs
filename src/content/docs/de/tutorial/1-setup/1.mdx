## Hol dir die Entwicklungswerkzeuge, die du brauchst

### Terminal

Du wirst eine **Befehlszeile (Terminal)** verwenden, um dein Astro-Projekt zu erstellen und wichtige Befehle auszuführen, um deine Website zu erstellen, zu entwickeln und zu testen.

Du kannst die Kommandozeile über ein lokales Terminalprogramm für dein Betriebssystem aufrufen. Gängige Programme sind **Terminal** (MacOS/Linux), **Command Prompt** (Windows) und **Termux** (Android). Eine dieser Anwendungen befindet sich wahrscheinlich bereits auf deinem Computer.

### Node.js

Damit Astro auf deinem System läuft, musst du außerdem [**Node.js**](https://nodejs.org/en/) in der Version `v18.17.1` oder `v20.3.0` oder höher installiert haben. (`v19` wird nicht unterstützt.)

Um zu prüfen, ob du bereits eine kompatible Version installiert hast, führe den folgenden Befehl in deinem Terminal aus:

```sh
node -v

// Beispiel Ausgabe
v18.17.1
```

Wenn der Befehl eine höhere Versionsnummer als `v18.17.1` oder `v20.3.0` (ohne `v19`) zurückgibt, bist du startklar!

Wenn der Befehl eine Fehlermeldung wie `Command 'node' not found` oder eine niedrigere Versionsnummer als die erforderliche zurückgibt, musst du [eine kompatible Node.js-Version installieren](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

### Code-Editor

Außerdem musst du einen **Code-Editor** herunterladen und installieren, um deinen Code zu schreiben. 

:::tip[Wir verwenden...]
In diesem Tutorial wird **VS Code** verwendet, aber du kannst jeden Editor für dein Betriebssystem benutzen.
:::

<Steps>
1. [Lade VS Code](https://code.visualstudio.com/#alt-downloads) oder einen anderen Code-Editor deiner Wahl herunter und installiere ihn. 
</Steps>
