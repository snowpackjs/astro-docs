<Lede> 在此教學中，你將透過從零開始建立一個功能齊全的部落格來學習 Astro 的主要功能，並將其部署到網路上！🚀 </Lede>

在過程中你將：
- 設置你的開發環境
- 為你的網站增加頁面和部落格文章
- 使用 Astro 元件進行建構
- 查詢和處理本地檔案
- 為你的網站增加互動性
- 部署你的網站

想要預覽一下你即將要建立的內容嗎？你可以到 [GitHub](https://github.com/withastro/blog-tutorial-demo) 或 [StackBlitz](https://stackblitz.com/github/withastro/blog-tutorial-demo/tree/complete?file=src%2Fpages%2Findex.astro) 去查看專案的最終成果。

:::note
如果你想要使用一個預先建立好的網站來探索 Astro，你可以到 https://astro.new 去挑選一個模板，然後在線上編輯器裡打開並編輯。
:::
