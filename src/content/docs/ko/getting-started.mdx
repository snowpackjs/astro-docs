<h2>Astro란 무엇인가요?</h2>

Astro는 **콘텐츠 중심의** **빠른** 웹 사이트를 구축하기 위한 **올인원(all-in-one)** **웹 프레임워크**입니다.

## 주요 기능

- **컴포넌트 아일랜드:** 더 빠른 웹사이트 구축을 위한 새로운 웹 아키텍처입니다.
- **서버 우선 API 설계:** 사용자의 장치에서 값비싼 수화를 제거합니다.
- **기본적으로 제로 JS:** 속도를 저하시키는 자바스크립트 런타임 오버헤드가 없습니다.
- **에지 지원:** Deno 또는 Cloudflare와 같은 글로벌 에지 런타임을 포함한 어디에나 배포할 수 있습니다.
- **사용자 정의 통합:** Tailwind, MDX 및 100개 이상의 다양한 통합 중에서 선택할 수 있습니다.
- **자유로운 UI 선택:** React, Preact, Svelte, Vue, Solid, Lit 등을 지원합니다.

Astro를 특별하게 만드는 요소가 궁금하다면 [왜 Astro인가](/ko/concepts/why-astro/)에서 자세한 분석을 확인하세요. ✨

## 첫 프로젝트 시작하기

`create astro` CLI 마법사를 사용해 로컬에서 새로운 Astro 프로젝트를 시작하세요!

<PackageManagerTabs>
  <Fragment slot="npm">
  ```shell 
  # npm으로 새 프로젝트 생성 
  npm create astro@latest 
  ```
  </Fragment>
  <Fragment slot="pnpm">
  ```shell 
  # pnpm으로 새 프로젝트 생성 
  pnpm create astro@latest 
  ```
  </Fragment>
  <Fragment slot="yarn">
  ```shell 
  # yarn으로 새 프로젝트 생성 
  yarn create astro 
  ```
  </Fragment>
</PackageManagerTabs>

[설치 가이드](/ko/install/auto/)에는 CLI 마법사를 사용하여 Astro를 설치하고, 기존 Astro GitHub 저장소에서 새 프로젝트를 생성하고, Astro를 수동으로 설치하는 방법에 대한 단계별 지침이 있습니다.

🧰 [사전 제작된 테마](https://astro.build/themes/)로 다음 프로젝트를 시작하세요.
